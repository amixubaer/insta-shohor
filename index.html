How does Javascript work?

Javascript is a programming language which handles a single task at a time. It is a client side scripting language. We don't have to compile the codes before sending to the browser. The codes passes through compiler which makes it machine redable. It is an interpreter language. It runs through javascript engine. Different browser has different engine such as google developed v8, mozila developed spider monkey etc.

What does javascript event loop do?

Javascript event loop is an amazing feature. It executes asynchronus tasks which is given to it. After completing the loop it become inactive untill next task. It allows to use callbacks and promises. With these callbacks or promisses it is assigned with new tasks. When Js interpreter recognizes functions like setTimeOut which would take some time to execute, Js interpreter sends them to web api and executes the next lines of the call stack.

What's the difference between local storage and session storage?

There are some difference between local storage and session storage. They are:-
1. The session storage is temporary but local storage is permanent.
2. The session storage destroys when user close the tab/browser. but the local storage remains until user delete it.
3. Local storage allows to keep value in the browser for long time where as session storage keeps the value only for one session.
4. Using local storage may cause some security issue where using session storage is safe.
